## 🚀 Tecnologias Utilizadas

- **HTML5**  
- **CSS3**  
- **Bootstrap**  
- **JavaScript**

## 📂 Estrutura do Projeto

- **index.html**: Página principal do portfólio.  
- **css/**: Arquivos de estilo personalizados.  
- **js/**: Scripts JavaScript para interatividade.  
- **assets/**: Imagens e outros recursos.
